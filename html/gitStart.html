GitHub Project Setup

Welcome to the first section where you will first setup your GitHub project. You will need to login to an account at: github.com and then you will need to follow these instructions to setup your first empty repository. Once you are logged into GitHub, click the following button to create your new repository.

Note: Always look at the red rectangles to see where the next step will be.

Once you open the next part you will see where you can add the title of your project. You can also see that you can add a description and set the project to public or private, I would suggest leaving them public for now. Finally, you have some options to add with the initialization of your respository, and we will leave those options blank for now as well.
Once you've completed the steps above, you should have a repository setup inside your GitHub account. You will want to use the copy button which should be highlighted in red. You'll want to manually copy the following command for the next course in this GitHub course. git remote add origin "Your GitHub website url". Head back to the main page to check this course complete and start the next course.
