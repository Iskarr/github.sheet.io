GitHub Localized Repository Setup

1

Welcome to the second section where you will initialize your GitHub project. You must be logged into your account. You will need have a folder you plan to setup this project in.
Open up VS Code and use the command
CTRL + O or CMD + O to open a folder that you want your project in. Once you have the project open you will need to open the terminal using the following command:
Control(^) key + tilda key (`) You should have something like this:

Note: Always look at the red rectangles to see where the next step will be.

1
2

Now that you have the terminal open, your project is open in VS code, its time to initialize your project locally.

(You will want to verify you have git installed by running this command:
git --version

if you get an error you might not have git installed, which means you will need to install it from this website: https://git-scm.com/

2
3
If everything is working from this point and you have a version of git showing up in your terminal, type in the following commands to locally set up and connect your project to your GitHub project in the cloud.
git init
git add .
git commit -m "Your message here"
git branch -M main
git remote add origin https://github.com/adonovan28/cool_project101.git

(Make sure to use your own link, check back into section one to find your link)

git push -u origin main
